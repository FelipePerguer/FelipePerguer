# 👩🏻‍💻 Felipe Perguer

**`Analista de dados e Automação`**

Me chamo Felipe Mazui Perguer, tenho 22 anos e sou natural do Rio Grande do Sul. Atualmente, curso Análise e Desenvolvimento de Sistemas na UNIFATECIE e trabalho com dados, área na qual sigo me desenvolvendo constantemente.
